# Stage 2 Jack Project

## What this stage contains

This stage is a simpler predecessor to the final stage3 app:
- two selectable shapes
- four-direction movement
- action counting per shape
- final summary on exit

There is no rotation or scaling here. That makes this stage useful as an incremental checkpoint and easier to explain in a walkthrough.

## Files

- `Main.jack`
- `ShapeA.jack`
- `ShapeB.jack`

## Controls

- `1` - select ShapeA
- `2` - select ShapeB
- Arrow keys - move the selected shape
- `Q` / `ESC` - quit

## Build and run

1. Copy the OS VM files from `nand2tetris/tools/OS/` into this folder.
2. Compile the folder with JackCompiler.
3. Load the VM output in the VM Emulator.
4. Use `No Animation` for the best rendering result.
